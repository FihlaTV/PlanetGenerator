Planetgenerator

Let's create a planet !

PlanetGenerator is software which randomly generater 3d planets. The aim of this project is to have a starting point, to create 3D environments, in which a planet is necessary (games, simulation tools, learning etc.).

Try the executable project.

You can try the executable project (on Windows). Linux users can build the project with make (make files come with the code). The executable project provides a graphical user interface to help setting all the fundamental parameters. Anyway the exexutable project doesn't provide all the features present in the code, since it has been thought just as starting point for those who want to work with this project. Download and run PlanetGenerator.

  1. Choose the size of the planet.
  2. Seed is a unique number which identifies the land shape. The same seed gives always the same land shape (exception for different operative systems).
  3. Percentage of surface covered by the water (or any liquid depending on the chosen color).
  4. Choose the color for water (liquid part), terrain and mountains.
  5. Choose frequency and amplitude (see simplex noise). It affects the shape of the planet.
  6. Preview of the 2D map of the planet.
  7. Set size and the kind of 2D map (eqirectangolar or cube) and save the image (.pmg)
  8. Import a .mesh file and put on the planet's surface.
  9. Export the planet in a mesh file.
