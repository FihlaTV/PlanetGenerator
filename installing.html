Planetgenerator

Let's create a planet !

PlanetGenerator is software which randomly generater 3d planets. The aim of this project is to have a starting point, to create 3D environments, in which a planet is necessary (games, simulation tools, learning etc.).

Getting started

This project has been built using C++, Ogre 3D and QT (you don't need this if you don't need the graphical user interface).
You can set up Ogre 3D for Visual Studio from here.
You can set up Ogre 3D for QtCreator from here.
You can set up QT for Visual Studio from here.
These are the guidelines we've used. Anyway you can follow any tool (for example a different IDE) or any tutorial you want. The setting up of the environment may change and we couldn't write anything more from what you get online.
